# Link Button
The `LinkButton` is just an extension of the [Button](?path=/docs/components-button--overview) component, so all applicable styles and properties from [Button](?path=/docs/components-button--overview) also apply for `LinkButton`.


## Usage
Use link button for redirecting the user to a different url: within the same app or external url.

Please use the `to` property for your url, no matter if it's internal or external link.

The component will parse the `to` property, and if it starts with `http://` or `https://` it will use an `a` element instead of `Link`.

### Internal Link
Uses react-router-dom's `<Link>` for linking when the `to` property is a relative url:
